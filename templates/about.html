📊 About This Project

AI-powered stock prediction system combining multiple machine learning models with real-time data visualization.

🤖 Models Used
  • ARIMA – Time series forecasting
  • LSTM – Deep learning for sequential data
  • Linear Regression – Trend estimation
  • Voting Ensemble – Combines all models
⚙️ Key Features
  • Real-time stock data integration
  • Interactive dashboard with charts
  • Multi-model prediction system
  • User authentication (login/signup)
📌 Project Overview

This project is designed to predict stock prices using multiple machine learning models. Instead of relying on a single algorithm, it combines predictions from ARIMA, LSTM, and Linear Regression using an ensemble approach to improve reliability.

The system fetches real-time stock data, processes it through these models, and displays the results through an interactive web dashboard. This makes it closer to real-world financial applications rather than just a theoretical model.
