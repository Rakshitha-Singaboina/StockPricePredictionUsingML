📘 Model Analysis Notebook

This section explains how different machine learning models are used for stock price prediction and how they perform.

📊 ARIMA

ARIMA is a statistical model used for time series forecasting. It captures trends and seasonality in stock prices.

🧠 LSTM

LSTM is a deep learning model that learns patterns from sequential data, making it suitable for stock prediction.

📈 Linear Regression

Linear Regression predicts stock trends based on historical price relationships.

⚙️ System Workflow
  • Fetch stock data using API
  • Preprocess and clean data
  • Apply ARIMA, LSTM, and Linear Regression
  • Combine predictions using Voting Ensemble
  • Display results in dashboard
🚀 Why Multiple Models?

Stock market data is highly volatile and unpredictable. Using a single model may lead to inaccurate predictions. By combining multiple models, the system improves reliability and reduces prediction errors.
